# Simple Skill Example

これは最小限の構成を持つシンプルなスキルの例です。

## 概要

このスキルは以下を示します：

- 必須の YAML frontmatter
- 基本的な Markdown 構造
- 最低限必要なセクション

## 使用方法

### ステップ 1: スキルの起動

Claude がこのスキルを自動的に検出し、適切なタイミングで使用します。

### ステップ 2: 指示に従う

スキルの指示に従って処理を実行します。

## 例

### 例 1: 基本的な使い方

**シナリオ**: シンプルなタスクを実行

**実行手順**:

1. タスクの内容を確認
2. 必要な処理を実行
3. 結果を確認

**ポイント**:

- シンプルな構造を維持
- 必要最小限の情報のみ提供

## まとめ

このスキルは最小限の例を示しています。より複雑なスキルについては、`complex-skill` の例を参照してください。
